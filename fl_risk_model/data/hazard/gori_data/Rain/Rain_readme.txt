Synthetic Rainfall

All rainfall is simulated with the Tropical Cyclone Rainfall Model (Zhu et al., 2013), which is a physics-based convective model that simulates rainfall occurring from 1) frictional convergence, 2) topographic effects, 3) vortex stretching, 4) baroclinic effects. 

ptot_mat  = contains total rainfall in mm for each county (rows) and each synthetic storm (columns). Counties are listed in the "US_counties.shp" file.
